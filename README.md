# Blind Assistant

This project is a real-time object detection application designed for visually impaired users. It uses a YOLOv5 model to detect objects and provides audio feedback using text-to-speech (TTS).

## Features
- Real-time object detection
- Audio feedback for detected objects
- Uses Gradio for easy web-based interface

## Installation

Clone the repository and install dependencies:

```bash
git clone <repository-url>
cd Blind-Assistant
pip install -r requirements.txt
